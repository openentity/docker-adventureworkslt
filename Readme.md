# AdventureWorksLT Database for Testing

Docker image of AdventureWorksLT sample database (MSSQL and Postgres). These images are published to Github Packages via GitHub Actions, you can pull and run the images with Docker Desktop:

```
docker pull ghcr.io/openentity/docker-adventureworkslt:mssql2022
docker pull ghcr.io/openentity/docker-adventureworkslt:postgres18
```
